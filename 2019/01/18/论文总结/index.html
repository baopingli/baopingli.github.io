论文总结

首先一切的基础都是ADMM的那本书《Proximal Algorithms》按着老师的读书笔记需要重新看一下。

FCSC 中是直接进行增广拉格朗日表示，然后求偏导交替迭代计算。

FFCSC 有的进行公式的泛化的表示，将目标函数表示成相同形式的几个子函数，表示成近端算子的形式，进行求解，这个还需要再看一下。

论文名称 特点
fast convolutional sprase coding 快速卷积稀疏编码 主要强调的是卷积，没有卷积的时候假设的是图像之间的像素是独立的没有关系的，但是事实上是存在关系的，所以说这篇论文主要是引入了卷积，将信号本身内部的关系考虑进来，然后使用ADMM算法分解成子问题进行求解。
fast and flexible convolutional sprase coding 首先增加了M矩阵，可以更好地处理边界条件，但是求解的时候使用了一种泛化的解法，就是flexible，将目标函数分解成了几个子函数然后进行求解。
frame-based image deblurring with unkonwn boundary condtions using the admm 这篇文章主要是对边界条件的处理，经典的边界条件已经不能很好的满足了，这篇文章主要的讲述增加了这个M矩阵之后怎么求解比较好。
scalable online convolutional sprase coding 这篇文章就比较吊了，online的操作，意味着内存小，计算快，具体的也记不太清了
TV相关的一些文章Restoration of Poissonian Images Using Alternating Direction Optimization TV主要用来进行去噪，使图片平滑，对于Dx整体运用正则化有点意义，可以使用admm来进行解决。
admm penalty parameter selection by Residual balancing 对增广拉格朗日中的系数rho的自适应的
consensus convolution sprase coding 共识，就是将大的分成小的部分然后去提取特征然后再合在一起这样一个过程，这篇文章效果比较好，当计算3D 4D这种在其他的算不动的情况下还能算动所以说效果比较好。有必要深挖。然后这个代码也是基于ffcsc的。对这个进行改进很有必要。
Optimization Methods for Convolutional Sparse Coding 这篇文章对卷积系稀疏编码的中用到的算法进行解释，怎么计算的这样的一个过程。

2.proximal algorithms 近端算法
